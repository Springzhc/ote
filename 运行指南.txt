# 出口管理系统 - 运行指南

由于PowerShell环境中批处理文件可能出现中文显示问题，建议按照以下步骤手动运行操作菜单：

## 步骤1：打开命令提示符
1. 按下 `Win + R` 键打开运行对话框
2. 输入 `cmd` 并按下回车键，打开命令提示符窗口

## 步骤2：切换到项目目录
在命令提示符中输入以下命令并按下回车键：
```
cd /d d:\Mycode\ote
```

## 步骤3：运行操作菜单
输入以下命令并按下回车键：
```
execute_all_steps.bat
```

## 步骤4：选择操作
在打开的菜单中，输入对应数字选择要执行的操作：
1. 安装Git
2. 上传代码到GitHub
3. 打包项目成EXE文件
4. 创建Windows安装程序
5. 退出

## 注意事项
- 确保所有批处理文件（install_git.bat、upload_to_github.bat、package_as_exe.bat）都在d:\Mycode\ote目录下
- 如果遇到任何问题，请检查文件是否存在并重试
